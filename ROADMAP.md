# Roadmap

## Now - active
- Expanding the component catalog (targeting 1000+ components across 60+ themes)
- Deepening metadata: `works_with`, `recommended_collections`, semantic tags
- MCP server for direct agent access without HTTP fetching

## Next
- Collections: curated component sets for common creative briefs (portfolio, artist showcase, editorial)
- Semantic search via embeddings over the component index
- Patterns layer: named aesthetic systems (glassmorphism, brutalist, editorial, generative)

## Later
- API endpoint for filtered queries
- Versioned component schema
- Community contributions
